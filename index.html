Agent Based Model:

About the Model: This Agent Based Model (ABM) was constructed as part of the Programming For Georaphical Information Analysis: Core Skills module, a component of the MSc Geographical Information Systems course, at The University of Leeds. The ABM acts as a simulation of the behaviour of agents ineracting within an environment. The ABM has a basic graphical user interface (GUI), allowing a user to run the model through the loop of iterations. The code for the ABM is provided via GitHub, at: 'Portfolio Repsository'
